City library is © IAAC, it's a digital interface developed by using computer vision and machine learning algorithms. This tool aims to detect waste materials in cities from building construction, treating buildings as material banks and urban environment as “mines”.
The term “Urban mining” is used to describe the process of recovering and reusing a city’s materials, which is the focus of this tool.
